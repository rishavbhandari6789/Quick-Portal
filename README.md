# Quiz Portal

**Quiz Portal** is a modern, interactive web application built with **React** and **Vite** that allows users to test their knowledge through engaging multiple-choice quizzes.
It is designed with a clean, responsive UI and a focus on speed, simplicity, and user experience.


## Overview

The project demonstrates a complete front-end workflow using **React hooks**, **state management**, and **modular component design**.
It is ideal for showcasing skills in React application development, logic handling, and clean UI/UX practices.


## Features

* Multiple-choice question-based quiz
* Real-time feedback on user selections
* Automatic score calculation and display
* Option to retake the quiz
* Fast development environment with **Vite**
* Fully responsive interface for mobile and desktop


## Tech Stack

| Technology             | Purpose                                             |
| ---------------------- | --------------------------------------------------- |
| **React 18+**          | Front-end UI library                                |
| **Vite**               | Development and build tool for lightning-fast setup |
| **JavaScript (ES6+)**  | Application logic and interactivity                 |
| **CSS / Tailwind CSS** | Styling and responsive design                       |



## Installation & Usage

```bash
# Clone the repository
git clone https://github.com/rishavbhandari6789/Quick-Portal.git

# Navigate into the project
cd quiz-portal

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```


## Why I Built This

This project was developed to explore how interactive educational tools can be built efficiently using modern JavaScript frameworks.
It also serves as a demonstration of best practices in **React component architecture**, **state management**, and **frontend responsiveness**.
